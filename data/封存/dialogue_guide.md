# RPG Maker 對話修改快速指南

為了確保遊戲對話顯示正常且不溢出，請在修改 `events` 中的 `code: 101` 與 `code: 401` 時遵循以下規則：

## 1. 代碼定義
- **`code: 101` (對話框設定)**：放置頭像、背景、位置。
- **`code: 401` (文字內容)**：實際顯示的對話字句。

## 2. 強制限制
- **每行長度 (Code 401)**：**每行最多不得超過 20 個中文字字**（包含標點符號）。
- **對話行數**：一個對話框建議最多放置 **4 行** `401`。
- **縮排 (Indent)**：`401` 的 `indent` 必須與前面的 `101` 完全一致。

## 3. 控制字元 (常用)
- `\\C[n]`：變更顏色。
- `\\V[n]`：變數內容。
- `\\N[n]`：角色名稱。

---
**範例**：
```json
{ "code": 101, "indent": 0, "parameters": ["People1", 0, 0, 2] },
{ "code": 401, "indent": 0, "parameters": ["這是一行不超過二十個這是一行不超過二十個"] },
{ "code": 401, "indent": 0, "parameters": ["字內容的範例對話。"] }
```
